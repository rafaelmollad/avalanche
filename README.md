## Tecnologías utilizadas

- React
- React Router
- Firebase
- React Hook Form
- Git y GitHub
- CSS
- SwiperJS

## Como ejecutar el proyecto localmente

### Descargar el proyecto

```
git clone https://github.com/rafaelmollad/avalanche.git
```

### Instalar node modules

```
npm install
```

### Abrir la aplicación

```
npm start
```
